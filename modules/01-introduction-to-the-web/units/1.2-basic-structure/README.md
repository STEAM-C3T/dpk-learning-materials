---

## Learning Objectives

By the end of this unit, students will be able to:

1. **Identify** the basic structure of an HTML document (doctype, html, head, body).
2. **Create** a simple HTML file with semantic tags.
3. **Explain** the purpose of key HTML elements (headings, paragraphs, lists, images).

---

## Overview

Unit 1.2 builds on the conceptual foundation from Unit 1.1 by introducing hands-on HTML coding. Students will:

- Write their first complete HTML document from scratch
- Use semantic tags (`<header>`, `<main>`, `<footer>`)
- Add headings, paragraphs, lists, and images
- Test their work in a browser

**Estimated Time:** 90–120 minutes
