## Learning Objectives

By the end of this unit, students will be able to:

1. **Explain** how the web works (clients, servers, browsers).
2. **Describe** the roles of HTML, CSS, and JavaScript.
3. **Identify** key web technologies and their purposes.

---

## Overview

Unit 1.1 provides a conceptual foundation for understanding how the web operates. Students will:

- Explore the client-server model
- Understand URLs, DNS, and HTTP
- Distinguish between HTML (content), CSS (style), and JavaScript (behavior)
- Recognize the browser as an interpreter of web code

**Estimated Time:** 45–60 minutes
