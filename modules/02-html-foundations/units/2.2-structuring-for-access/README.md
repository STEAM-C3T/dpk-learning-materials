## Learning Objectives

By the end of this unit, students will be able to:

1. **Build** forms with accessible label-input associations.
2. **Create** data tables with proper headers and captions.
3. **Understand** semantic structure for accessibility.
4. **Test** pages with keyboard navigation.

---

## Overview

Unit 2.2 teaches students to build accessible forms and tables. Students will:

- Create forms with properly associated labels
- Use various input types (text, email, select, textarea)
- Build data tables with headers and captions
- Test accessibility with keyboard navigation

**Estimated Time:** 45–60 minutes
