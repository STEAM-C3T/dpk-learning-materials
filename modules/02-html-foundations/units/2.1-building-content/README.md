## Learning Objectives

By the end of this unit, students will be able to:

1. **Use** headings to create clear document hierarchy.
2. **Add** paragraphs, lists, links, and images with proper markup.
3. **Write** descriptive alt text for images.
4. **Structure** content semantically for accessibility.

---

## Overview

Unit 2.1 focuses on building well-structured, accessible content using semantic HTML elements. Students will:

- Create heading hierarchies (`<h1>` through `<h6>`)
- Add paragraphs, ordered and unordered lists
- Write descriptive link text and alt text
- Understand the importance of semantic structure

**Estimated Time:** 45–60 minutes
