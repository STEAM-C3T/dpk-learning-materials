# Module 02: HTML Foundations — Learning Materials

**Focus:** Content hierarchy, accessibility  
**Example Outcome:** Student topic/portfolio page

---

## Module Overview

This module builds on Module 01 by teaching students to create well-structured, accessible HTML content. Topics include heading hierarchies, lists, links, images with alt text, and semantic markup.

**Prerequisites:** Module 01 (Introduction to the Web)  
**Estimated Total Time:** 2–3 hours (across 2 units)
