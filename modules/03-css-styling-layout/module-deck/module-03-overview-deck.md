# Module 03: CSS Styling & Layout

Digital Proficiency Kit
Web Development for STEAM

---

## Purpose & Placement

- Builds on: Module 02 — HTML Foundations
- Leads to: Module 04 — JavaScript Essentials
- Duration: 2–3 hours

**Key Artefacts:**

- Styled portfolio page
- Responsive grid layout

---

## Learning Outcomes

- Apply CSS selectors and properties
- Control typography, color, and spacing
- Build responsive layouts with Flexbox/Grid
- Evaluate contrast and readability

---

## Teacher Focus Points

- Unit 3.1: 45–60 min
- Unit 3.2: 45–75 min
- Projects: 30–45 min
- Common misconceptions: Inline styles vs. stylesheets; absolute units vs. responsive
- Accessibility: Contrast, focus states, text sizing

---

## Big Ideas

- CSS separates content from presentation
- Responsive design adapts to screen sizes
- Accessibility requires sufficient contrast

Diagram:

```
HTML (structure) + CSS (style) = Webpage
```

## Accessibility

- Check text/background contrast (WCAG AA)
- Ensure visible keyboard focus
- Allow browser zoom (avoid fixed sizes)

---

## Extension

- Add responsive navigation
- Create print stylesheet
- Use CSS custom properties (variables)
