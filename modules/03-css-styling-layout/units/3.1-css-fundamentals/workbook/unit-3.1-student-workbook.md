## Glossary

| Term                | Definition                               | Example                                     |
| ------------------- | ---------------------------------------- | ------------------------------------------- |
| Selector            | Targets HTML elements for styling        | `p`, `.class`, `#id`                        |
| Property            | Aspect of element to style               | `color`, `font-size`, `margin`              |
| Value               | Setting for a property                   | `red`, `16px`, `10px auto`                  |
| Box model           | Content, padding, border, margin layers  | See diagram in tutorial                     |
| Hex color           | Color code using hexadecimal             | `#3498db`                                   |
| RGB                 | Color defined by red, green, blue values | `rgb(52, 152, 219)`                         |
| HSL                 | Color by hue, saturation, lightness      | `hsl(204, 70%, 53%)`                        |
| External stylesheet | Separate .css file linked to HTML        | `<link rel="stylesheet" href="styles.css">` |
