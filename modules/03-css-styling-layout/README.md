# Module 03: CSS Styling & Layout — Learning Materials

**Focus:** Typography, colours, responsive design  
**Example Outcome:** Styled and responsive site

---

## Module Overview

This module introduces CSS for visual design and responsive layouts. Students learn typography, colour theory, spacing, and modern layout techniques (Flexbox/Grid) while maintaining accessibility (contrast, focus).

**Prerequisites:** Modules 01–02 (HTML foundations)  
**Estimated Total Time:** 3–4 hours (across 2 units)
